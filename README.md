# Eco55 🌱

## AI-Based Environmental Incident Reporting Android Application

Eco55 – Sahyadri Samrakshane is an AI-powered Android application designed to help users report environmental incidents such as waste dumping, water leakage, pollution, road damage, and other public environmental issues in real time.

The application integrates Firebase cloud services and Google Maps API to provide accurate location-based incident reporting with a modern Android UI.

---

# 📱 Project Overview

Eco55 aims to create a smarter and cleaner environment by enabling citizens to quickly report environmental issues directly from their mobile devices.

The system stores reports in Firebase Realtime Database and allows easy management of incident information using cloud-connected infrastructure.

# 🏗 System Architecture

```text
User → Android App → Firebase Database
                    ↘ Google Maps API
```

The application collects environmental incident details from the user, attaches location data using Google Maps API, and stores the report securely in Firebase.

# ⚙️ Installation Guide

## Step 1: Clone Repository

```bash
git clone https://github.com/sanjayb23/Eco55-sahyadri-samrakshane.git
```

---

## Step 2: Open in Android Studio

* Launch Android Studio
* Click Open Project
* Select the project folder

---

## Step 3: Sync Gradle Files

Allow Gradle to download dependencies automatically.

---

## Step 4: Configure Firebase

* Create Firebase Project
* Add Android application
* Download `google-services.json`
* Place it inside:

```text
app/google-services.json
```

---

## Step 5: Add Google Maps API Key

Add your Google Maps API key inside:

```xml
AndroidManifest.xml
```

---

## Step 6: Run Application

* Connect Emulator or Android Device
* Click Run ▶ in Android Studio
